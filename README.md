## Installation

Requirements:

- <a href="https://nodejs.org/en/download/package-manager/">Node.js v14.17.5 / npm v6.14.14 >=</a>

- <a href="https://bower.io/#install-bower">Bower installed globally</a>

```
$ bower i
$ firefox index.html # launch with your favorite browser
```
